Proste metody, które na pewno Ci się przydadzą! W celu wywołania funkcji rysującej używamy metody getContext(), w celu narysowania prostokątu używamy fillRect(), a aby zadeklarować gradient liniowy: createLinearGradient().

Korzystając z języka JavaScript, można rysować na elemencie Canvas var example = document.getElementById('example'); var context = example.getContext('2d'); context.fillStyle = "rgb(255,0,0)"; context.fillRect(30, 30, 50, 50); Powyższy kod rysuje na ekranie czerwony prostokąt.

Przykładowy fragment kodu HTML5 wykorzystujący tag . Jeżeli tag ten nie jest obsługiwany, wyświetli się wpisany tekst. Twoja przeglądarka nie obsługuje elementu Canvas.

Aby utworzyć gradient liniowy w canvasie używamy: var c = document.getElementById("myCanvas"); var ctx = c.getContext("2d"); // Create gradient var grd = ctx.createLinearGradient(0,0,200,0); grd.addColorStop(0,"red"); grd.addColorStop(1,"white"); // Fill with gradient ctx.fillStyle = grd; ctx.fillRect(10,10,150,80);
